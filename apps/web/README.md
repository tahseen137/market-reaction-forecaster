# Web App Placeholder

The intended frontend stack for this repo is:

- Next.js
- TypeScript
- Tailwind CSS
- TanStack Query
- ECharts for time-series and forecast visualizations

Recommended initial pages:

- `/` event feed and watchlists
- `/events/[id]` event detail and evidence trace
- `/forecasts/[id]` scenario forecast with confidence bands
- `/backtests` replay and calibration dashboard
- `/portfolio` paper portfolio and exposure view
